OLVIDÉ MI CONTRASEÑA

Para recuperar la contraseña de la aplicación debe:

1. Proporcionar su Usuario y correo electrónico.
2. Presione el botón 'Obtener código'. ...Se enviará un código de seguridad al correo proporcionado para permitir el cambio de clave.
3. Ingrese el código de seguridad y la nueva contraseña y presione en botón 'Cambiar Contraseña'"
